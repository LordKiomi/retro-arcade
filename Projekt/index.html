🎮 Retro Game Project

Snake, Pac-Man i menu w klimacie automatów arcade

Projekt przedstawia zestaw kultowych gier przeglądarkowych: Snake i Pac-Man, opakowanych w spójne, stylowe menu w klimacie retro arcade. Celem było stworzenie wciągającej prezentacji z użyciem wyłącznie HTML, CSS i JavaScript – bez frameworków i z dużą dbałością o klimat lat 80/90.

Neonowe przyciski, pikselowa czcionka i tło przypominające stare ekrany CRT to nie tylko estetyka – to powrót do klasyki gier, które bawiły nas zanim poznaliśmy open-worldy i ray tracing.
